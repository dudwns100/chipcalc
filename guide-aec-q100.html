AEC-Q100 완전 가이드

차량용 집적회로의 국제 표준 신뢰성 인증.
Grade 0~4 등급부터 PPAP 프로세스까지 실무자 관점에서 정리.

📅 2026년 기준⏱ 읽기 약 8분🔧 실무 체크리스트 포함
AEC-Q란? Q-series 등급 Grade 0~4 테스트 항목 인증 프로세스 PPAP IATF 16949 비용·기간

AEC-Q란 무엇인가?

AEC(Automotive Electronics Council)에서 제정한 자동차용 전자부품의 신뢰성 시험 표준입니다.

AEC(Automotive Electronics Council)는 1993년 Chrysler · Ford · GM의 미국 3대 완성차 업체가 공동 설립한 컨소시엄입니다. 자동차에 쓰이는 전자부품이 극한 환경(온도·진동·습도·EMC)에서도 안정적으로 동작하도록 공통된 신뢰성 시험 기준을 만들어 왔습니다.

차량용 반도체 공급업체는 고객 완성차 업체에 납품하려면 해당 제품이 이 AEC-Q 인증을 통과했다는 Qualification Report를 제공해야 합니다.

⚠ AEC-Q는 법적 의무가 아니지만 사실상 필수

AEC-Q 인증이 법으로 강제되는 것은 아니지만, 완성차 OEM · Tier 1 업체가 AEC-Q 인증 부품만 채택합니다. 인증 없이는 사실상 차량용 시장 진입이 불가능합니다.

AEC-Q 패밀리 — 부품 종류별 표준

부품 유형에 따라 다른 표준을 적용합니다.

표준대상 부품주요 예시
AEC-Q100Integrated Circuit (IC)MCU, SoC, PMIC, ADC, 센서 IC
AEC-Q101Discrete Semiconductor트랜지스터, 다이오드, MOSFET, IGBT
AEC-Q102Optoelectronic (LED)자동차 LED 조명, 광센서
AEC-Q103MEMS Sensor가속도계, 자이로, 마이크로폰
AEC-Q104Multichip Module (MCM)멀티칩 패키지, SiP
AEC-Q200Passive Component저항, 커패시터, 인덕터, 크리스탈

💡 가장 많이 쓰이는 건 AEC-Q100

이 가이드는 주로 AEC-Q100 (IC용)을 다룹니다. ChipCalc의 원가 계산기는 AEC-Q100 기반 Die 원가를 계산합니다.

AEC-Q100 Grade 0~4 — 작동 온도 등급

차량 내 부품 위치에 따라 요구되는 온도 범위가 다르며, 이게 Grade를 결정합니다.

Grade작동 온도 범위차량 내 위치예시 칩
Grade 0-40°C ~ +150°C엔진/파워트레인 옆엔진 ECU MCU, 트랜스미션 제어 IC
Grade 1-40°C ~ +125°C엔진룸 (보드 장착)Powertrain MCU, 파워 IC
Grade 2-40°C ~ +105°C실내 가혹 부위Body ECU, ADAS SoC
Grade 3-40°C ~ +85°C실내 표준인포테인먼트 SoC, Gateway
Grade 40°C ~ +70°C일반 (소비자급과 유사)일부 인포테인먼트 부품

⚠ Grade가 높을수록 단가는 올라갑니다

Grade 0은 Grade 3에 비해 보통 1.5~3배 이상 단가가 비쌉니다. Die 설계 변경, 별도 테스트, 고온 패키지 필요 등 추가 비용이 들어가기 때문입니다.

AEC-Q100 주요 테스트 항목

전기적·환경적·기계적 스트레스 시험을 모두 통과해야 합니다.

Group A — 가속 환경 스트레스

  • HTOL (High Temperature Operating Life) — 고온 1000시간 동작
  • HTSL (High Temperature Storage Life) — 고온 보관 1000시간
  • TC (Temperature Cycling) — 급격한 온도 변화 1000 cycle
  • THB (Temperature Humidity Bias) — 85°C/85% RH, 1000시간

Group B — 가속 수명

  • HAST (Highly Accelerated Stress Test) — 130°C/85% RH
  • AC (Autoclave) — 121°C 고압 포화 증기

Group C — 패키지 무결성

  • PC (Pre-Conditioning) — MSL 시뮬레이션
  • Solderability — 납땜성
  • PD (Physical Dimensions) — 외형 치수

Group D~E — 전기적/기계적 특성

  • ESD (Electrostatic Discharge) — HBM ±2kV, CDM ±500V
  • Latch-Up — 래치업 내성
  • EMC — 전자기 적합성
  • 기계적 충격·진동 — 자동차 환경 모사

인증 획득 프로세스

계획부터 최종 보고서 제출까지 보통 6~12개월 소요됩니다.

  1. Design for Automotive — 설계 단계부터 AEC-Q100 고려 (온도 마진, ESD 보호)
  2. Wafer/Package 준비 — 인증용 샘플 3 lot 이상 생산
  3. Test Plan 작성 — AEC-Q100 Rev-H 기준 시험 항목 확정
  4. Stress Test 수행 — 외부 공인 시험소 또는 사내 시험소
  5. 결과 분석 · 실패 시 재설계 — Root cause 분석, DOE 반영
  6. Qualification Report 작성 — 고객사 제출용 최종 보고서
  7. Production Part Approval (PPAP) — 완성차 OEM 승인

PPAP — 양산 부품 승인 프로세스

AEC-Q100 인증 이후 실제 고객사 양산 승인까지의 마지막 관문입니다.

PPAP(Production Part Approval Process)는 자동차 산업에서 공급업체가 양산 부품을 납품하기 전 완성차 OEM에 제출해야 하는 표준 승인 프로세스입니다. 18개의 표준 문서를 제출해야 하며, 가장 엄격한 Level 4~5에서는 수개월이 걸립니다.

PPAP Level제출 문서 범위일반적 사용
Level 1Warrant만 제출저위험 변경
Level 2Warrant + 샘플 + 제한적 데이터경미한 변경
Level 3 (기본)Warrant + 샘플 + 전체 데이터대부분의 신규 부품
Level 4고객 지정 문서중요 부품
Level 5전체 현장 검증 포함고위험·신기술 부품

IATF 16949 — 품질 경영 시스템

AEC-Q100이 '제품' 인증이라면, IATF 16949는 '회사 전체' 품질 시스템 인증입니다.

IATF 16949(International Automotive Task Force)는 ISO 9001을 기반으로 자동차 산업에 특화한 품질경영시스템(QMS) 표준입니다. 차량용 반도체를 공급하는 회사는 기본적으로 이 인증을 보유하고 있어야 고객사와 계약이 가능합니다.

✅ 실무 체크

완성차 OEM/Tier 1이 공급업체를 선정할 때 기본 요구사항: ① IATF 16949 인증 보유 (회사) ② AEC-Q100 통과 (제품) ③ PPAP 제출 (부품별). 이 세 가지가 차량용 반도체 진입의 "3종 세트"입니다.

인증 비용과 소요 기간

처음 AEC-Q100 획득에는 최소 6개월, 수만 달러~수십만 달러의 비용이 듭니다.

항목소요 기간대략 비용
설계 변경 (자동차급)2~6개월Die 재설계 비용에 포함
샘플 준비 (3 lot)2~4개월웨이퍼 · 패키지 비용
외부 공인 시험3~6개월$20,000 ~ $100,000
시험 실패 시 재시험3~6개월 추가추가 $20,000 이상
PPAP 승인 (고객별)2~6개월문서 · 감사 비용

💰 한 번 인증받은 부품은 가격이 유지됩니다

AEC-Q100 인증 비용이 크기 때문에 차량용 반도체는 7~15년 장기 공급이 전제됩니다. 이게 차량용 반도체가 소비자용보다 프리미엄 가격을 유지할 수 있는 주요 이유입니다.
