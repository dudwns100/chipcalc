3
분석 시작

미리보기 확인 후 "분석 시작"을 누르면 차트와 통계가 자동 생성됩니다.

필수 컬럼
  • 제품명 — 식별자
  • 실구매가 — 분석 기준값 (Y축)
  • 연간수량 — 인하여력 계산용
  • X축 지표 — 성능을 대표하는 숫자 컬럼 최소 1개
  • 추정원가 — Hybrid / Should-cost 분석용 (권장)
선택 컬럼
  • 시리즈 — 없으면 제품명에서 자동 추출
  • 기타 숫자 컬럼 — 자유롭게 추가 (예: 전류, 전압, Flash 등) → X축에서 선택 가능

💡 컬럼 순서는 상관없습니다. 헤더 이름으로 자동 인식합니다.

💡 헤더는 한국어/영어 모두 지원 (제품명/Product, 실구매가/Actual 등).

💡 반도체 외 다른 부품 분석에도 사용 가능합니다.
