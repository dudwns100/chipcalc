개인정보처리방침

Privacy Policy · 최종 수정일: 2026년 4월

본 방침은 한국어와 영어로 작성되었습니다. 법적 효력은 한국어 버전을 기준으로 합니다.

1. 수집하는 정보

  • Google 로그인 시: 이름, 이메일 주소 (로그인 식별 목적). 비밀번호는 Google이 관리하며 ChipCalc에 전달되지 않습니다.
  • 계산 저장 시: 사용자가 💾 저장 버튼을 직접 누른 계산 결과만 Supabase(클라우드 DB)에 저장됩니다.
  • Google Analytics: 페이지 조회수, 방문 시간, 브라우저 유형 (익명화). 개인을 특정하지 않습니다.
  • localStorage: 패키지 단가, 언어 설정, 다크모드 설정. 서버로 전송되지 않습니다.

2. 데이터 보호

  • 저장된 계산 기록은 RLS(Row Level Security)로 보호되어 본인만 조회·삭제 가능합니다.
  • 서비스는 HTTPS를 통해 암호화된 연결로 제공됩니다.
  • 행동 추적, 광고 타겟팅, 제3자 데이터 판매 없음.

3. 쿠키 및 추적 기술

Google Analytics 쿠키(_ga, _gid, _gat)가 익명 방문자 통계에 사용됩니다. 브라우저 설정에서 쿠키를 비활성화하거나 Google Analytics 옵트아웃 도구를 사용할 수 있습니다.

4. 제3자 서비스

  • Google Analytics — 방문자 통계 (Google LLC, 미국)
  • Supabase — 로그인·데이터 저장 (Supabase Inc., 미국) · 개인정보방침
  • GitHub Pages — 웹 호스팅 (GitHub Inc., 미국)
  • Anthropic Claude API — 반도체 제품 검색 기능. 검색어가 Anthropic 서버로 전송됩니다.

5. 방침 변경

본 방침은 변경될 수 있으며, 변경 시 이 페이지에 최신 버전을 게시합니다.
